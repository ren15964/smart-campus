## 项目整体结构

```
bishe/
├── smart-campus-backend/      # 后端项目（Spring Boot）
├── smart-campus-frontend/     # 前端项目（Vue 3）
├── database.sql               # 数据库建表脚本
├── 智慧校园平台需求文档.md    # 需求文档
└── 项目结构说明.md            # 本文档
```

## 主要技术栈

### 后端
- Spring Boot 2.7+
- MyBatis Plus 3.5+
- Spring Security + JWT
- MySQL 8.0
- Redis
- Lombok
- Hutool

### 前端
- Vue 3
- Vue Router 4
- Pinia
- Element Plus / Ant Design Vue
- Axios
- Vite
- Sass
